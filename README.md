# Custom Mod

Custom mod for Wayward. Work in progress.

## Setup

```powershell
npm install
npm run build
```

The mod is linked into the game at `Wayward/mods/custom-mod` (junction → this folder).

## Development

```powershell
npm run watch
```

Enable **Custom Mod** in the in-game Mod Manager, then load a save to verify (`Custom Mod loaded.` in logs).
